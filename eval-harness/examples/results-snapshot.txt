The eval ran successfully — no crash. The harness loaded the eval file, ran the
agent for each case, and applied all three graders. The deterministic graders
(length+topic, content-exclusion) produced normal PASS/FAIL results visible in
the GRADERS column. Only the model-graded grader failed, because it requires an
API key to call Claude for qualitative assessment.

This is graceful degradation: you can see how the deterministic graders behave
before you have an API key configured. Once you set the key, all three graders
will run and you'll get a complete picture.

To fix:

  Windows PS:     $env:ANTHROPIC_API_KEY="sk-ant-..."
  Windows CMD:    set ANTHROPIC_API_KEY=sk-ant-...

Then re-run:
  npm test
  — or —
  npm run eval -- --file my-evals/my-agent-eval.ts
